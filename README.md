# SBRI Converter



> 서울특별시 노선정보 조회 서비스 API 결과 값 변환기



## Installation

```sh
pip install -r requirements.txt
```



## Usage example

```sh
python3 main.py
```



- 본 프로그램에 입력된 API 주소는 저장되지 않습니다.
- 본 프로그램에서는 API service Key를 검증할 수 없습니다. 따라서 요청 실패 시 Url을 다시 한 번 확인해주세요.
